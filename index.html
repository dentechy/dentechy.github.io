Dennis Feliciano

Dennis Feliciano is very passionate about nutrition, fitness and technology. Over the years, he became very knowledgeable in all three. In college he studied biochemistry, learning the fundamentals of just how the body processes food, down to a molecular level. While in college, he also studied and passed a certification examination to become an ACSM certified personal trainer, acquiring useful knowledge along the way on individualized fitness programs and coaching strategies. In grad school, he learned even more about the biochemical aspects of nutrition, such as the metabolic changes that occur during the postprandial state.

The journey with technology though, began several years ago when he first installed Linux on an old Sony Vaio laptop. Over the years, he would experiment with the command line and eventually enroll in an online course tailored towards Linux system administration. That course provided even more valuable information on managing my own Linux systems, but doing everything strictly through the command line, even editing files through vim. His current Linux systems include a raspberry pi configured as an offsite backup in CT and another that is used as a local file server.

The progression to web development was only natural from there. He has since enrolled in a full stack web development program, starting with the basics in HTML, CSS and JavaScript and moving to more advanced concepts such as Node.js, Express, mongodb and React. Raspberry Pi File Server Tutorial

One of the first projects Dennis created directly involved his passion for Linux. Essentially a wireframe website was created using HTML and CSS, with an original article explaining step by step how to create a raspberry pi file server, all from the command line. The main prerequisites for following this tutorial are the hardware itself (a raspberry pi, hard drive and network) and access to the terminal from within the raspberry pi.

When this file server is setup properly, it is able to share files with most major operating systems, including Windows, mac OS, Linux, Android and iOS. Content such as movies can also be streamed over the local network. View tutorial here
